neovim下的fzf

安装fzf.vim

配置fzf

当我在vim中按下”\“，vim会调用fzf查找你需要编辑的文件，选中文件后回车即可编辑

当我在vim中按下”Ctrl+h”，vim会显示你最近打开的文件

当我在vim中按下”Ctrl+l”，vim会高亮包含你搜索关键词的一行代码

当我在vim中按下”Ctrl+b”，vim会显示当前你打开的buffer，输入数字或者符号即可跳转buffer
